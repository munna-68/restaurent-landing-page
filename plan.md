### Gap / Padding
- **Letter spacing**: All text uses 0% letter spacing
- **Line spacing**: Body uses AUTO; headings use fixed PIXELS or PERCENT values
- **Image stroke gap**: 3px strokes on room images; 4px strokes on gallery images
- **Shadow offset**: 10px x 10px on card images, 4px x 4px on circles

## 7. Layout & Grid

### Page Dimensions
- **Thai Restaurant (Desktop - 1)**: 1440 px wide × 3742 px tall
- **Resort (Desktop - 2)**: 1440 px wide × 5813 px tall
- **Mobile (iPhone 14 - 1)**: 390 px wide × 768 px tall

### Layout Pattern
- **Full-width sections**: Every major section spans the full 1440 px width
- **Layout direction**: Sections stack vertically (flex column on page level)
- **Content alignment**: Text blocks are left-aligned within ~521–587 px wide containers
- **Image placement**: Images are offset to the right or left of text columns
- **No visible grid columns**: The design uses a single-column stacked layout with offset image regions

### Notable Layout Measurements
- Navbar height: 73 px
- Text content width: approximately 521–587 px
- Image card: 556 × 377 px (or similar)
- Button dimensions: 133 × 48 px (Thai), 118–126 × 41–44 px (Resort)
- Three image cards in menu section: each 361 × 454 px, spaced horizontally

### Breakpoints
- There is a separate mobile artboard (iPhone 14 - 1, 390 × 768 px)
- No additional breakpoints visible in the design

## 8. Component Inventory

### Thai Restaurant Design ("Desktop - 1")

1. **Navbar** — Top bar (1440 × 73 px, background #F9EFE7). Contains brand logo text "Thai Phuket Restaurant" on the left, nav links (Menu, About us, Contact) center-aligned, and "Book a Table" button (133 × 48 px, background #883E40, white text). Layout: flex row.

2. **Hero Section** — Full-width white background (1440 × 684 px). Left side contains heading "An Ambient Dining Experience" (Abril Fatface 68.4px), description text, tagline, and two CTA buttons (gold #D2B753 and dark red #883E40). Right side has a masked image group (591 × 599 px) with decorative clipping. Layout: flex row with text left, image right.

3. **About / Feature Section** — Background #F9EFE7 (1440 × 684 px). Contains heading "The Finest Thai Cuisine", description text, and a dark red "Menu" button on the left. Right side has a large image (686 × 946 px). Layout: flex row.

4. **Tradition Section** — White background (part of Rectangle 21, 1440 × 1001 px). Heading "Tradition & Family Recipes" with description and "View Menu" button on the left. Right side has image placeholder (627 × 480 px). Layout: flex row.

5. **Menu / Signature Dishes Section** — Three menu item cards (each 361 × 454 px) arranged horizontally: Khao Pad, Pad Thai, Tom Yum Goong. Each has an image placeholder (#D9D9D9 background) and dish name (Abril Fatface 68.4px). Layout: flex row with 3 columns, gap ~425 px between cards.

6. **Testimonials Section** — Background #F8F8F8 (1440 × 417 px). Contains quote "My favorite Thai place in Geneva!" (Roboto SemiBold 39px, color #883E40), body text, star rating (5 star icons), and author "Mateusz Madura, Geneva, Switzerland". Layout: flex column, centered.

7. **Footer** — Background #F9EFE7 (1440 × 267 px). Contains brand logo "Thai Phuket Restaurant" on the left, nav links (Menu, About us, Gallery, Contact) and copyright. Layout: flex row.

### Resort Design ("Desktop - 2")

8. **Navbar (Resort)** — Top bar (1440 × 73 px, background #BAD1C2 at 75% opacity). Logo "Aurelia" left, nav links (Restaurant, Home, About us, Contact), "Book a Room" button (118 × 41 px, background #4FA095, border-radius 8px). Layout: flex row.

9. **Hero Section (Resort)** — Background #F6F6C9 (1440 × 684 px). Tagline "Setting the standard for luxury everywhere" (Patua One 50px), heading "Inspired by local traditions..." (Roboto SemiBold 36.2px). Right side has image card with drop shadow (10×10, blur 20, rgba(0,0,0,0.25)). Layout: flex row.

Three feature cards: "Dive into history at the Venetian Castle", "Tick water sports off your bucket list", "Relax at the boutique spa and club". Each has icon + heading (Roboto SemiBold 25.8px) + description (Roboto Condensed Italic 18.3px). Layout: 3-column flex row.

11. **Rooms / Cottages Section** — Background #F6F6C9 (1440 × 829 px). Heading "Cottages that blend comfort & elegance" (Patua One 68.4px). Three room cards with images (strokes 3px black), each showing "Standard Room" (Patua One 35px), price "$ 230" (Roboto Slab 35px) + "/night" (Roboto ExtraLight 24.8px), amenities label "4 beds rooms", and "Check availability" CTA link. Layout: flex column with heading + 3-column card row.

12. **Amenities Bar** — Inline amenity labels with icons: Pool, Spa, Fishing, Dining, 24/7 Bar, Barbeque (Roboto Medium ~16px each). Flex row, evenly spaced.

13. **Dining Section** — Background #FAF7F0 (1440 × 1126 px). Heading "Fresh local ingredients, delicious global cuisine." (Patua One 50px), body text (Roboto 17.7px). Two image cards side by side with drop shadows. Layout: flex row.

14. **Testimonials Section (Resort)** — Background #F6F6C9 (1440 × 1078 px). Heading "Word on the street" (Roboto SemiBold 39px). Testimonial text and author attribution "— Isabella Miller" (Roboto Condensed Thin Italic 28px). Image carousel/gallery grid with 5 images showing rooms and dining, each with 4px black stroke. Layout: flex column.

15. **Footer (Resort)** — Background #BAD1C2 (1440 × 267 px). Logo "Aurelia" (Concert One). Nav links: Rooms, About us, Gallery, Offers, Spa. Copyright notice. Layout: flex row.
